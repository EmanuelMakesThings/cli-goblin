# Changelog

All notable changes to the **cliGoblin** project will be documented in this file.

## [1.0.1] - 2026-01-07
### Changed
- **Creature Art Fix**: Renamed "Rust Crab" to "Rust Bunny" because the ASCII art was clearly a rabbit. The memory safety remains uncompromising.
- **Documentation**: Updated `GUIDE.md` to reflect the creature name change.
- **Version**: Bumped package version to 1.0.1.

## [1.0.0] - 2026-01-07
### Added
- Initial release of **cliGoblin**.
- Core logic: Hunger, Mood, Energy, Health, Bond stats.
- **Interactive Menu**: `cligoblin menu` for settings and tools.
- **Developer Mode**: Hidden menu (`GobbyDev`) for stat manipulation and time travel.
- **Creature Hunting**: `hunt` command to capture ASCII creatures.
- **Inventory System**: View and rename captured creatures.
- **Live Watch**: `watch` command for real-time animations.
- **Reactive Dialogue**: Responds to `git`, `rm`, time of day, and neglect.
